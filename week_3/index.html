Week 3: kinetic sculpture

Rotate till green project

Introduction

Objectives of this project is to create a kinetic sculpture based on a DC motor. One of the possibilities is to create an energy converter- mechanically identical to an electric motor, but operates in the reverse direction, converting mechanical energy into electrical energy. The faster DC motor is rotating, the more current it will generate.
For demonstration purposes LED indicators are used to visualize the speed of rotation. This is a very simple idea but involves skills previously learned in this class: Arduino programming to measure the input voltage and turn on LEDs, and laser cutting to create an enclosure for the device.

Measuring DC motor output voltage using arduino

Arduino allows to use serial port to visualize the input signal from the DC motor (Tools menu - Serial Plotter). That allows to adjust a code and circuit design to receive a proper input signal. The input signal from the DC motor has been measured - it covers the range from 0 to 3.5-4 volts.

Building the Circuit

  • The circuit is based on Arduino Nano: https://store.arduino.cc/usa/arduino-nano
  • It consists of 3 LEDs connected via 1kOhm resistors to the A1-A3 analog outputs.
  • DC motor will be connected to A0 input via 1 mF Capacitor

The capacitor has been added to reduce the input voltage spikes. On the left the plot shows input voltage plot with a capacitor connected in parallel to the DC motor. On the right - without the capacitor. Capacitor smoother the input voltage spikes.

writing a code

General idea for the program is to change LED blink delay with respect to the potential input of DC motor.
Arduino boards contain a multichannel, 10-bit analog to digital converter. This means that it will map input voltages between 0 and the operating voltage(5V for Arduino Nano) into integer values between 0 and 1023. The reasonable voltage that could be generated by the DC motor is about 800-850.
‍
C++ allows to map input voltage to any value by using the the map() function.

Code example: Read sensor value from DC motor input. If it is between 50 to 450, the red LED blinking speed will be changing. The higher the value, the faster LED will blink. LED blinking delay value linearly mapped between 300 and 10 linearly to the sensor reading value.
