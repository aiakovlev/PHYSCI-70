Sensor calibration

The set-up has 4 phototransistors (1080-1019-ND) serving as a direction setting inputs.

2 for forward/backward move and 2 for rotation of the rover.

IMPORTANT:

To trigger the sensor, UV laser pointer should be used. UV light makes the piece of paper to fluoresce producing intensive blue light.

The blue light intensity is enough to trigger the phototransistor output.

Phototransistor and white paper envelope

UV Laser pointer shining on the paper envelope.

Sensor reading: value more that 10 triggers the direction inputs.

Adjusting Stepper Library

Two 28BYJ-48 unipolar stepper motors are used.

The default arduino Stepper library didn'w work well with this motot because of incorrect steps sequence declaration.

The following code in Stepper library has been adjusted to make this motor work


switch (thisStep) {
  case 0:  // 1010
	digitalWrite(motor_pin_1, HIGH);
	digitalWrite(motor_pin_2, LOW);
	digitalWrite(motor_pin_3, HIGH);
	digitalWrite(motor_pin_4, LOW);
  break;
  case 1:  // 0110
	digitalWrite(motor_pin_1, LOW);
	digitalWrite(motor_pin_2, HIGH);
	digitalWrite(motor_pin_3, HIGH);
	digitalWrite(motor_pin_4, LOW);
  break;
  case 2:  //0101
	digitalWrite(motor_pin_1, LOW);
	digitalWrite(motor_pin_2, HIGH);
	digitalWrite(motor_pin_3, LOW);
	digitalWrite(motor_pin_4, HIGH);
  break;
  case 3:  //1001
	digitalWrite(motor_pin_1, HIGH);
	digitalWrite(motor_pin_2, LOW);
	digitalWrite(motor_pin_3, LOW);
	digitalWrite(motor_pin_4, HIGH);
  break;
}

To this one:


// CODE FOR 28BYJ-48 motor

Final design

  • Power bank provides power supply to all motors. Arduino Nano is connected to a cumputer.

  • ItsyBitsy used just for taking 5V pin from the power bank

  • Phototransistor is connected via 1kOhm resistors.

  • Arduino A0, A1, A3, A4 inputs read data from phototransistors.
