3. RUNNING a "blink" PROGRAM

void setup() {
// initialize digital pin LED_BUILTIN as an output.
pinMode(13, OUTPUT);
}
// the loop function runs over and over again forever
void loop() {
digitalWrite(13, HIGH); // turn the LED on (HIGH is the voltage level)
delay(1000); // wait for a second
digitalWrite(13, LOW); // turn the LED off by making the voltage LOW
delay(1000); // wait for a second
}

During uploading of the "Blink" sketch the following error message appeared:

To fix that you need to make sure that the correct board and port are selected. In my case the board is Adafruit ItsyBitsy M4, port COM3.

4. BOOTLOADER

During uploading procedure another error occurred - IDE stopped at sketch uploading process.

This could be fixed by entering into bootloader mode.

To do that you need to press reset button twice. After that Arduino will be recognized as an external USD drive. Try to repeat upload procedure.

In case the above instruction will give no results, you can try to update bootloader. Follow these instructions.

5. LED run

void setup() {
pinMode(13, OUTPUT);
pinMode(12, OUTPUT);
pinMode(11, OUTPUT);
pinMode(10, OUTPUT);
}

void loop() {
for (int i = 13; i >= 10; i--) {
digitalWrite(i, HIGH);
delay(250);
digitalWrite(13, LOW);
digitalWrite(12, LOW);
digitalWrite(11, LOW);
digitalWrite(10, LOW);
}
}
